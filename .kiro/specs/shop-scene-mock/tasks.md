- [x] 2. ショップの状態管理を実装する
  - [x] 2.1 基本状態クラスを作成する

  - [x] 2.2 商品データモデルを定義する

    - ProductData レコードを定義する（商品名、アイコンパス、価格、通貨種別、商品種別、毛糸量）
    - CurrencyType と ProductType の列挙型を定義する
  - [x] 2.3 モックデータリストを実装する
    - ガチャリスト、アイテム商品リスト、毛糸パック商品リストを保持する
    - 毛糸の消費・追加メソッドを実装する
  - _Requirements: 11.1, 11.2, 11.3, 11.4, 11.5, 11.6_

- [x] 3. ショップのUIビュー構造を構築する
  - [x] 3.1 メインビュー構造を作成する
    - ShopView クラスを作成し、タブボタン（ItemButton, PointButton）をバインドする

  - [x] 3.2 タブ切り替えUIを実装する

    - 選択中タブの視覚的なスタイル変更を実装する
    - デフォルトでアイテムタブを選択状態にする
  - [x] 3.3 コンテンツ領域を準備する
    - アイテムタブ用コンテンツ（_itemContent）を用意する
    - 毛糸タブ用コンテンツ（_pointContent）を用意する
    - タブより下のCategories領域のみスクロール可能にする
  - _Requirements: 2.1, 2.2, 2.3, 2.4, 2.5, 2.6_

- [x] 4. セルビューコンポーネントを実装する
  - [x] 4.1 ガチャセルビューを作成する

  - [x] 4.2 商品セルビューを作成する
    - ProductCellView クラスを作成し、アイコン、商品名、価格をバインドする
    - セルタップイベントを定義する
    - アイテム用と毛糸パック用で共通利用する
  - [x] 4.3 セルのインタラクティブ状態管理を実装する
    - 毛糸不足時のボタン無効化メソッドを実装する
    - 購入不可を示す視覚的フィードバックを実装する
  - _Requirements: 4.1, 4.2, 4.4, 4.5, 4.6, 5.1, 5.2, 6.2, 6.3, 8.6, 9.3_

  - [x] 5.2 タブ切り替えロジックを実装する

  - [x] 5.3 セルコンテンツ設定を実装する

- [x] 6. シーンUIの手動配置とレイアウト設定を行う
  - [x] 6.1 カテゴリ構造を配置する

    - LayoutElement で各セルのサイズを設定する
  - [x] 6.4 VContainer統合を完了する
